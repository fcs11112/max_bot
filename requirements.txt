# MAX bot without Watbot

Этот проект заменяет Watbot на обычный Python-код.

Схема:

```text
MAX -> Railway -> FastAPI -> PPTX + Excel -> PDF -> ZIP -> ссылка пользователю
```

## Что есть в проекте

- `main.py` - backend, MAX-бот, меню, шаблоны, генерация PDF.
- `app.py` - совместимость с запуском `uvicorn app:app`.
- `requirements.txt` - Python-библиотеки.
- `Procfile` - команда запуска Railway.
- `nixpacks.toml` - установка LibreOffice на Railway для конвертации PPTX в PDF.
- `Dockerfile` - более надежный вариант сборки: Python + LibreOffice + шрифты для кириллицы.

## Переменные Railway

Открой Railway -> проект -> `Variables` и добавь:

```text
BOT_TOKEN=токен_бота_из_MAX
BASE_URL=https://адрес-твоего-проекта.up.railway.app
WEBHOOK_SECRET=любая_строка_без_пробелов_например_mysecret12345
```

`WEBHOOK_SECRET` можно придумать самому. Главное: английские буквы, цифры, `_` или `-`.

## Подключение webhook

После деплоя открой в браузере:

```text
https://адрес-твоего-проекта.up.railway.app/setup_max_webhook
```

Если всё хорошо, MAX ответит примерно:

```text
MAX response 200:
{"success":true}
```

## Новый сценарий бота

Пользователь пишет `/start`.

Бот показывает:

```text
Выбери действие:
```

Кнопки:

- `Мои шаблоны`
- `Сгенерировать`
- `Инструкция`

### Мои шаблоны

Бот показывает список загруженных шаблонов:

```text
1. template.pptx
2. sales.pptx
```

Кнопки:

- `Добавить шаблон`
- `Удалить шаблон`
- `Главное меню`

При добавлении бот просит отправить `.pptx`.

При удалении бот просит отправить цифру шаблона из списка.

### Сгенерировать

1. Бот показывает список шаблонов.
2. Пользователь отправляет цифру нужного шаблона.
3. Бот просит отправить Excel `.xlsx` или `.xls`.
4. После Excel бот просит выбрать падеж:
   - Именительный
   - Родительный
   - Дательный
   - Винительный
   - Творительный
   - Предложный
5. Бот генерирует PDF-файлы, складывает их в ZIP и отправляет ссылку.

## Как должен выглядеть PPTX

В шаблоне должны быть текстовые плейсхолдеры через проценты, которые совпадают с колонками Excel.

Пример текста в PPTX:

```text
%ФИО%
%КЛАСС%
%ДАТА%
%НОМИНАЦИЯ%
```

Пример колонок Excel:

```text
%ФИО% | %КЛАСС% | %ДАТА% | %НОМИНАЦИЯ%
Иванов Иван Иванович | 5А | 13.05.2026 | Лучший проект
Петрова Анна Сергеевна | 6Б | 13.05.2026 | Победитель
```

Каждая строка Excel станет отдельной грамотой.

## Падежи ФИО

Склонение применяется к колонкам:

- `%ФИО%`
- `%Фамилия Имя Отчество%`
- `%Фамилия%`
- `%Имя%`
- `%Отчество%`
- `%Полное имя%`

Для русского языка используется бесплатная библиотека `pymorphy3`.

Важно: автоматическое склонение может ошибаться на редких, иностранных или необычно написанных ФИО.

## Важно про PDF

`python-pptx` умеет создавать PPTX, но не умеет сохранять PDF. Поэтому проект использует LibreOffice:

```text
PPTX -> LibreOffice headless -> PDF
```

Для Railway добавлены `Dockerfile` и `nixpacks.toml`. После загрузки этих файлов обязательно сделай redeploy.
